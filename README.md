# UCSB Directory

Retrieves information from the UCSB directory.

## Installation

```
npm install ucsb-directory
```

## Usage

```javascript
var ucsb = require('./ucsb-directory');

ucsb('logan franken').then(function(result) {
  console.log(result);
});
```
